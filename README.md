# jpashop

----------------
##1. skills
----------------
1) environment
  - JDK 11

2) language
  - Java
  - Javascript
  
3) template 
  - thymeleaf 3.0
  - bootstrap 4.6.x

4) libraries
  - spring-boot
  - spring-web-mvc
  - spring-boot-starter-devtools
  - junit4
  - spring-boot-starter-data-jpa
  - p6spy
  - lombok
  - spring-boot-starter-validation

5) database
  - h2 database

----------------
##2. functions
----------------
1) 회원 등록
2) 회원 목록 조회
3) 상품 등록
4) 상품 목록 조회
5) 상품 정보 수정
6) 주문 등록
7) 주문 목록 조회(주문 회원별, 주문 상태별)
8) 주문 취소
